## Description
これは電気回路第２章演習問題９のP点における電圧値の変化を計算するプログラムです。 <br>
P点からみた時の各線路について <br>
&nbsp;受端側の反射係数をai <br>
$nbsp;送端側の反射係数をbi,透過係数をci <br>
としています <br>
<br>
各関数fiは、受端側からきた波を与えると透過してP点の電圧をあげ、各線路に反射または透過した波を発生させる再帰的関数になっています。 <br>
<br>
<br>
設計してないので計算量が半端なく、大体計算回数n=20~21くらいが僕の環境では妥当な感じでした。 <br>
